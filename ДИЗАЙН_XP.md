# Дизайн в стиле Windows XP

Проект полностью переработан в стиле Windows XP с характерными элементами интерфейса.

## Основные элементы дизайна

### Цветовая схема
- **Фон**: Синий градиент (#5a7fdc → #3e5faa) - классический фон рабочего стола XP
- **Панели**: Бежевый градиент (#ece9d8 → #d8d5c8) - цвет окон и кнопок
- **Хедер**: Синий градиент (#0997ff → #06f) - цвет заголовков окон
- **Текст**: Черный (#000000) на светлом фоне, белый на синем

### Кнопки в стиле XP
- Градиентный фон (бежевый)
- 3D эффект с помощью границ:
  - Светлые границы сверху и слева (#ffffff)
  - Темные границы снизу и справа (#808080)
- При нажатии границы инвертируются
- Тени для объема

### Окна и панели
- Бежевый градиентный фон
- 3D рамки с эффектом выпуклости
- Тени для создания глубины
- Без скругленных углов (border-radius: 0)

### Скроллбары
- Стилизованы под Windows XP
- Бежевый цвет с градиентом
- Кнопки прокрутки
- Серые границы

### Шрифты
- Tahoma - основной шрифт Windows XP
- Жирный текст для заголовков и кнопок
- Тени для текста на цветном фоне

## Обновленные компоненты

1. **Главная страница** - синий фон XP, бежевые панели
2. **Хедер** - синий градиент заголовка окна XP
3. **Форма авторизации** - окно в стиле XP с 3D эффектами
4. **Кнопки** - классические кнопки XP с градиентами

## Особенности

- Все элементы имеют угловатую форму (без border-radius)
- Используются классические цвета Windows XP
- 3D эффекты через границы и тени
- Ностальгический дизайн 2001 года

## Следующие шаги

Для полного соответствия стилю XP нужно обновить:
- Sidebar (боковая панель)
- TopBar (верхняя панель приложения)
- Player (музыкальный плеер)
- TrackCard (карточки треков)
- Страницы приложения (upload, favorites)

Все эти компоненты будут переработаны в стиле Windows XP с бежевыми панелями, 3D эффектами и классическими элементами интерфейса.
