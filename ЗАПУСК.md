# Инструкция по запуску проекта

## Проблема с .next папкой

Если возникает ошибка `EPERM: operation not permitted, open '.next\trace'`, выполните следующие шаги:

### Шаг 1: Закройте все процессы Node.js
1. Откройте Диспетчер задач (Ctrl + Shift + Esc)
2. Найдите все процессы `Node.js JavaScript Runtime`
3. Завершите все процессы Node.js

### Шаг 2: Удалите папку .next
1. Откройте проводник Windows
2. Перейдите в папку `D:\Diplomka\music-distribution`
3. Найдите папку `.next`
4. Удалите её полностью (Shift + Delete для безвозвратного удаления)

### Шаг 3: Запустите проект
Откройте терминал в папке проекта и выполните:

```bash
npm run dev
```

## Альтернативный способ

Если проблема повторяется, попробуйте:

1. Закройте VS Code / Kiro полностью
2. Удалите папку `.next` через проводник
3. Откройте проект заново
4. Запустите `npm run dev`

## После успешного запуска

Откройте браузер и перейдите по адресу: http://localhost:3000

## Структура проекта

- `/` - Главная страница с хедером
- `/auth` - Страница входа/регистрации
- `/app` - Главная страница приложения с музыкой
- `/app/upload` - Страница загрузки музыки
- `/app/favorites` - Страница избранных треков

## Навигация в хедере

- **Прослушать музыку** - переход к библиотеке треков
- **Выложить музыку** - форма загрузки новых треков
- **Понравившиеся треки** - избранное
- **Вход** - авторизация (если не авторизован)
- **Регистрация** - создание аккаунта (если не авторизован)
- **Имя пользователя** - профиль (если авторизован)
- **Выйти** - выход из аккаунта (если авторизован)
