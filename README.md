# MusicStream - Музыкальный стриминговый сервис

Полнофункциональный музыкальный стриминговый сервис по типу Яндекс.Музыки или Spotify, построенный на Next.js с TypeScript.

## Возможности

### Авторизация и регистрация
- ✅ Локальная регистрация пользователей
- ✅ Вход в систему
- ✅ Хранение данных в localStorage
- ✅ Защищенные маршруты

### Музыкальный плеер
- ✅ Воспроизведение треков
- ✅ Пауза/продолжить
- ✅ Следующий/предыдущий трек
- ✅ Управление громкостью
- ✅ Прогресс-бар
- ✅ Очередь воспроизведения

### Интерфейс
- ✅ Боковая панель с навигацией
- ✅ Главная страница с треками и альбомами
- ✅ Карточки треков с обложками
- ✅ Фиксированный плеер внизу экрана
- ✅ Поиск (UI готов)
- ✅ Адаптивный дизайн

### Дизайн
- ✅ Темная тема в красно-черных тонах
- ✅ Градиенты и анимации
- ✅ Hover-эффекты
- ✅ Современный UI/UX

## Технологии

- **Next.js 15** - React фреймворк с App Router
- **TypeScript** - типизация
- **CSS Modules** - стилизация
- **Context API** - управление состоянием
- **localStorage** - хранение данных

## Установка и запуск

```bash
# Перейдите в папку проекта
cd music-distribution

# Если папка .next вызывает проблемы, удалите её вручную через проводник

# Установите зависимости (если еще не установлены)
npm install

# Запустите сервер разработки
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Использование

### 1. Регистрация
- Перейдите на страницу `/auth`
- Нажмите "Зарегистрироваться"
- Введите имя, email и пароль (минимум 6 символов)
- Нажмите "Зарегистрироваться"

### 2. Вход
- Перейдите на страницу `/auth`
- Введите email и пароль
- Нажмите "Войти"

### 3. Прослушивание музыки
- После входа вы попадете на главную страницу `/app`
- Нажмите на любую карточку трека для воспроизведения
- Используйте плеер внизу экрана для управления

### 4. Навигация
- **Главная** - популярные треки и альбомы
- **Поиск** - поиск музыки (в разработке)
- **Моя библиотека** - ваши плейлисты (в разработке)

## Примечания

- Все данные хранятся локально в localStorage браузера
- Аудиофайлы используют моковые данные (URL пустые)
- Для полноценной работы нужно добавить реальные аудиофайлы
- Изображения обложек загружаются с Unsplash

## Решение проблем

Если возникает ошибка с `.next/trace`:
1. Закройте все процессы Node.js
2. Удалите папку `.next` вручную через проводник Windows
3. Запустите `npm run dev` снова
