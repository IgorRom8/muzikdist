# Интеграция с Jamendo API

## Что такое Jamendo?

Jamendo - это платформа с **бесплатной легальной музыкой** под открытыми лицензиями (Creative Commons). Все треки можно легально использовать в вашем приложении.

## Преимущества:

- ✅ **Легально** - вся музыка под открытыми лицензиями
- ✅ **Бесплатно** - API бесплатный для некоммерческого использования
- ✅ **Большая библиотека** - более 500,000 треков
- ✅ **Качественная музыка** - от независимых артистов со всего мира
- ✅ **Прямые ссылки на MP3** - можно воспроизводить напрямую

## Настройка:

### Шаг 1: Регистрация

1. Перейдите на https://devportal.jamendo.com/
2. Нажмите "Sign Up" и создайте аккаунт
3. Подтвердите email

### Шаг 2: Создание приложения

1. Войдите в Developer Portal
2. Нажмите "Create a new application"
3. Заполните форму:
   - Application name: "Music Distribution Platform"
   - Application type: "Web Application"
   - Description: "Music streaming platform"
4. Нажмите "Create"

### Шаг 3: Получение Client ID

1. После создания приложения вы увидите **Client ID**
2. Скопируйте его

### Шаг 4: Настройка .env

Откройте файл `.env` и замените:

```env
NEXT_PUBLIC_JAMENDO_CLIENT_ID=your_jamendo_client_id
```

На ваш реальный Client ID:

```env
NEXT_PUBLIC_JAMENDO_CLIENT_ID=abc123def456
```

### Шаг 5: Перезапуск сервера

```bash
npm run dev
```

## Использование:

После настройки:

1. Перейдите на страницу **"Открыть новое"** в меню
2. Вы увидите популярные треки из Jamendo
3. Можно воспроизводить, добавлять в плейлисты
4. Все треки легальны и бесплатны!

## Функции:

- **Популярные треки** - топ треков по популярности
- **Поиск** - можно искать треки (функция уже готова в коде)
- **По жанрам** - фильтрация по жанрам (функция готова)
- **Прямое воспроизведение** - MP3 файлы воспроизводятся напрямую

## Лимиты API:

- **Бесплатный план**: 10,000 запросов в день
- **Коммерческий план**: от €99/месяц для коммерческого использования

Для некоммерческого проекта бесплатного плана более чем достаточно!

## Альтернативы:

Если Jamendo не подходит, можно использовать:

1. **Free Music Archive** - https://freemusicarchive.org/
2. **ccMixter** - http://ccmixter.org/
3. **Incompetech** - https://incompetech.com/
4. **YouTube Audio Library** - бесплатная музыка от YouTube

## Важно:

⚠️ **Yandex Music API** не предоставляет публичный доступ и использование неофициальных методов может нарушать их условия использования.

Jamendo - это легальная и безопасная альтернатива с качественной музыкой!
